Materi Machine Learning memperkenalkan konsep inti, jenis pembelajaran, dan contoh penerapan.

## Apa itu Machine Learning?

Machine learning adalah cabang kecerdasan buatan yang membuat sistem belajar dari data tanpa diprogram secara eksplisit. Model mempelajari pola, membuat prediksi, atau mengklasifikasikan informasi baru.

## Jenis Machine Learning

- **Supervised learning**: model dilatih dengan data berlabel.
- **Unsupervised learning**: model menemukan pola tanpa label.
- **Reinforcement learning**: model belajar melalui umpan balik dan tindakan.

## Siklus pembelajaran

1. Kumpulkan data
2. Pra-pemrosesan dan fitur
3. Latih model
4. Evaluasi hasil
5. Deploy model ke aplikasi nyata

## Contoh aplikasi

- Klasifikasi teks dan gambar
- Rekomendasi produk
- Deteksi anomali dan prediksi

## Catatan
