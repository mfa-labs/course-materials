# Cloud Computing - 14 Pertemuan Kurikulum

## Pertemuan 1: Pengenalan Cloud Computing
**Tujuan:** Memahami konsep dasar cloud computing
- Definisi dan konsep cloud computing
- Sejarah perkembangan cloud computing
- Manfaat dan keuntungan cloud computing
- Tantangan dan risiko cloud computing
- Model penyebaran cloud (Public, Private, Hybrid, Community)

## Pertemuan 2: Cloud Service Models
**Tujuan:** Memahami berbagai model layanan cloud
- Infrastructure as a Service (IaaS)
- Platform as a Service (PaaS)
- Software as a Service (SaaS)
- Perbandingan ketiga model
- Use cases untuk setiap model

## Pertemuan 3: Cloud Providers dan Layanan Utama
**Tujuan:** Mengenal cloud provider utama
- AWS (Amazon Web Services)
- Microsoft Azure
- Google Cloud Platform (GCP)
- Perbandingan fitur dan harga
- Hands-on: Membuat akun cloud

## Pertemuan 4: Virtual Machines dan Compute Services
**Tujuan:** Memahami layanan komputasi cloud
- Konsep virtual machines
- Elastic Compute Cloud (EC2) di AWS
- Jenis instance dan sizing
- Scaling dan Load Balancing
- Praktik: Launch dan configure VM

## Pertemuan 5: Storage Services
**Tujuan:** Memahami layanan penyimpanan cloud
- Object Storage (S3, Cloud Storage)
- Block Storage (EBS, Persistent Disks)
- File Storage (EFS, Cloud Filestore)
- Database as a Service
- Praktik: Mengupload dan manage storage

## Pertemuan 6: Networking dalam Cloud
**Tujuan:** Memahami infrastruktur jaringan cloud
- Virtual Private Cloud (VPC)
- Subnets dan routing
- Security Groups dan Firewalls
- Load Balancers dan CDN
- Praktik: Configure VPC dan security

## Pertemuan 7: Database Services
**Tujuan:** Memahami layanan database cloud
- Relational databases (RDS, Cloud SQL)
- NoSQL databases (DynamoDB, Firestore)
- Data warehouse (Redshift, BigQuery)
- Backup dan disaster recovery
- Praktik: Setup dan query database

## Pertemuan 8: Identity and Access Management (IAM)
**Tujuan:** Memahami keamanan dan kontrol akses
- Users, Groups, dan Roles
- Policies dan permissions
- Multi-factor authentication
- Best practices untuk security
- Praktik: Configure IAM policies

## Pertemuan 9: Monitoring, Logging, dan Performance
**Tujuan:** Memahami operasi dan observabilitas cloud
- Monitoring tools (CloudWatch, Stackdriver)
- Logging dan log analysis
- Performance metrics
- Alerting dan notifications
- Praktik: Setup monitoring dashboard

## Pertemuan 10: Auto Scaling dan Orchestration
**Tujuan:** Memahami automation dalam cloud
- Auto scaling policies
- Container orchestration (Docker, Kubernetes)
- Infrastructure as Code (IaC)
- Terraform dan CloudFormation
- Praktik: Deploy containerized application

## Pertemuan 11: Serverless Computing
**Tujuan:** Memahami serverless architecture
- Function as a Service (FaaS)
- AWS Lambda, Google Cloud Functions
- Event-driven architecture
- Cost optimization dengan serverless
- Praktik: Deploy serverless function

## Pertemuan 12: Backup, Disaster Recovery, dan Business Continuity
**Tujuan:** Memahami reliability dan resilience
- Backup strategies
- Disaster recovery planning
- RTO dan RPO concepts
- Multi-region deployment
- Praktik: Implement backup solution

## Pertemuan 13: Cost Management dan Optimization
**Tujuan:** Memahami billing dan cost optimization
- Cloud pricing models
- Reserved instances dan savings plans
- Cost monitoring tools
- Optimization strategies
- Praktik: Analyze dan optimize costs

## Pertemuan 14: Cloud Security dan Compliance
**Tujuan:** Memahami security best practices
- Cloud security frameworks
- Data encryption at rest dan in transit
- Compliance standards (HIPAA, PCI-DSS, GDPR)
- Security auditing
- Case study dan best practices
