# Pertemuan 13: Cost Management dan Optimization

## Tujuan Pembelajaran
Setelah menyelesaikan pertemuan ini, mahasiswa diharapkan dapat:
- Memahami cloud pricing models
- Menggunakan cost management tools
- Mengimplementasikan cost optimization
- Melakukan cost forecasting
- Membuat cost optimization recommendations

## Materi Pembelajaran

### 1. Cloud Pricing Models
- On-demand pricing
- Reserved instances (1-year, 3-year)
- Spot instances
- Savings plans
- Dedicated hosts
- Data transfer costs
- Pricing by region

### 2. Cost Components
- Compute (EC2, Lambda, Container services)
- Storage (S3, EBS, EFS)
- Data transfer
- Database (RDS, DynamoDB)
- Networking (NAT, VPN, CloudFront)
- Add-on services

### 3. Cost Management Tools
- AWS Cost Explorer
  - Usage filtering
  - Cost analysis
  - Forecasting
- AWS Budgets
  - Budget creation
  - Alerts
  - Budget actions
- AWS Cost Anomaly Detection
- AWS Compute Optimizer

### 4. Cost Optimization Strategies
- Right-sizing instances
- Reserved instances vs on-demand
- Spot instances untuk non-critical workloads
- Scheduled scaling
- Unused resources cleanup
- Storage optimization (lifecycle policies, compression)
- Data transfer optimization

### 5. Tagging Strategy
- Cost allocation tags
- Mandatory tags
- Tag governance
- Tag-based cost allocation

### 6. Optimization by Service
- EC2 optimization
- Storage optimization
- Database optimization
- Lambda optimization
- Networking optimization
