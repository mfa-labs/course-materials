# Pertemuan 3: Cloud Providers dan Layanan Utama

## Tujuan Pembelajaran
Setelah menyelesaikan pertemuan ini, mahasiswa diharapkan dapat:
- Mengenal cloud provider utama di dunia
- Membandingkan fitur dan harga cloud providers
- Memahami layanan utama dari setiap provider
- Membuat akun cloud
- Mengeksplorasi cloud console/dashboard

## Materi Pembelajaran

### 1. AWS (Amazon Web Services)
- Sejarah dan market share
- Layanan utama: EC2, S3, RDS, Lambda
- Pricing model dan calculator
- Global regions dan availability zones
- AWS Management Console overview

### 2. Microsoft Azure
- Sejarah dan keunggulan
- Layanan utama: Virtual Machines, App Service, SQL Database
- Integration dengan Microsoft ecosystem
- Pricing dan Azure Calculator
- Azure Portal overview

### 3. Google Cloud Platform (GCP)
- Sejarah dan fokus teknologi
- Layanan utama: Compute Engine, App Engine, BigQuery
- Keunggulan di data analytics dan ML
- Pricing model
- Google Cloud Console overview

### 4. Perbandingan Cloud Providers
- Feature comparison matrix
- Pricing comparison (compute, storage, data transfer)
- Global presence dan compliance
- Vendor lock-in considerations
- Customer support dan documentation

### 5. Cloud Provider Terkecil/Alternatif
- IBM Cloud, Oracle Cloud, DigitalOcean
- Specialized providers
- Hybrid solutions
